## About
I’m a PhD candidate in Physics at **Purdue University** working in the **CMS Collaboration**, with research spanning **CERN** and **Fermilab**.  
My focus is on **Higgs pair production (HH → bbττ)** in Run-3 data, and on **efficient ML for real-time / low-latency detector environments**.

---

## Research Focus
- **CMS Run-3 Di-Higgs (HH → bbττ)**  
  Event reconstruction + selection pipelines, performance/efficiency studies, and trigger-level/scouting-style workflows.
- **Trigger / real-time constraints**  
  Methods that respect tight latency/throughput budgets (online-like selection, fast inference, resource-aware design).
- **Hardware-aware ML for HEP**  
  Structured pruning and quantization, digitization-aware training, and end-to-end co-design for deployment.

---

## Technical Stack
- **HEP:** ROOT / PyROOT, RDataFrame, histogramming + statistical workflows  
- **ML:** PyTorch, TensorFlow/Keras, model compression (pruning/quantization), profiling & benchmarking  
- **Systems:** Linux, Git, Docker, HPC-style workflows (batch processing, reproducibility, monitoring)
